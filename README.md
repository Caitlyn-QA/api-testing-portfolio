# API Testing Portfolio

## Overview

This repository demonstrates API testing using Postman and REST APIs.

The project covers:

- GET requests
- POST requests
- PUT requests
- DELETE requests
- Status code validation
- Response body validation
- Response time validation
- Environment variables
- Postman collections
